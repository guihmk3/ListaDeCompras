# ?? Gerenciador de Lista de Compras

Este é um programa simples desenvolvido em **C# com Windows Forms** para gerenciamento de listas de compras.  
O objetivo é oferecer uma interface prática e intuitiva para adicionar, visualizar e excluir itens de uma lista.

---

## ? Funcionalidades

- ? Adicionar itens à lista por meio de um **TextBox**  
- ? Visualizar os itens adicionados em um **ListBox**  
- ? Excluir **um único item selecionado**  
- ? Limpar toda a lista com apenas um clique  
- ? Interface amigável e fácil de usar
